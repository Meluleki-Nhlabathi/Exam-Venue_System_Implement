# Exam Venue Allocation System

A Java EE web application for managing exam venues, computers, lecturers, students, and student allocations for exams — built as a NetBeans project using EJB modules and JSP.

## Project Structure

- **AdminEJBModule** — EJB business logic layer (facades and entities for Admin, Computer, Exam, Lecturer, Module, Student, StudentAllocation, Venue, etc.)
- **Exam-Venue Allocation System** — Web application module (JSP pages for admin, lecturer, and student views)

## Tech Stack

- Java EE (EJB, JPA)
- JSP
- NetBeans project structure (Ant build)
